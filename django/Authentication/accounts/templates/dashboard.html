🔑 Behind the Scenes — Kya hua jab tune login kiya?

1
Form Submit
Login form ne POST request bheji → Django ne request.POST['username'] aur request.POST['password'] pakde
2
Password Verify
authenticate() ne DB mein stored hashed password se compare kiya. Sahi tha toh User object return kiya.
3
Session Bani
login(request, user) ne ek unique Session ID banaya, DB mein save kiya, aur browser ko cookie diya.
4
Protected Page Access
Har request pe browser wo cookie bhejta hai → @login_required check karta hai → valid hai toh page milta hai, nahi toh login pe redirect.
